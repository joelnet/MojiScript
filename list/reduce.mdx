# list/reduce

`reduce :: Function -> Any -> Iterable`

Returns a single item by iterating through the list, successively calling the iterator function and passing it an accumulator value and the current value from the array, and then passing the result to the next call.

An `Iterable` could be an `Array`, an `Iterator`, or `async Iterator`, or a `Functor`.

## Array Example

```javascript
import { reduce } from 'mojiscript'

const add = x => y => x + y
const array = [ 1, 2, 3 ]

reduce (add) (0) (array) //=> 6
```

## Iterator Example

```javascript
import { range, reduce } from 'mojiscript'

const add = x => y => x + y

reduce (add) (0) (range (1) (4)) //=> 6
```

# Async Example

```javascript
import { pipe, reduce, sleep } from 'mojiscript'

const add = x => y => x + y
const asyncAdd = x => pipe ([
  sleep (1000),
  add (x)
])

const array = [ 1, 2, 3 ]

reduce (asyncAdd) (0) (array) //=> Promise (6)
```
